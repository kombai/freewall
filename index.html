Free Wall

Creating dynamic grid layouts without gaps.

What is Freewall?

Freewall is a responsive and cross-browser jQuery plugin for creating dynamic grid layouts for desktop, mobile and tablet. Freewall is all-in-one solution help you to create many types of grid layout (flexible layout, grid layout, images layout, pinterest-like layout, etc) with nice CSS3 animation effects and call back events.

Some example layouts:

  • Flex layout

  • Grid layout

  • Images layout

  • Pinterest layout

How it works

Based on the width (or height) of a container and the width (height) of a cell unit, It will create a virtual matrix. Scaning the matrix at each cell will find a free cell around to made a free area, then try to fit a block element into it. In case no block can fit the gap, it will resize the block to fill the gap (that is one of the options).

Getting started

Freewall requires jQuery framework. It requires browsers which support CSS3 for some nice animation effects.

							
							    <!DOCTYPE html>
							    <html>
							    <head>
							      <title> freewall demo getting started</title>
							      <script src="js/jquery-1.7.min.js"></script>
							      <script src="js/freewall.js"></script>
							      <style type="text/css">
							        #container {
							          width: 80%;
							          margin: auto;
							        }
							        .item {
							          background: rgb(135, 199, 135);
							          width: 320px;
							          height: 320px;
							        }
							      </style>
							    </head>
							    <body>
							      <ul id="container">
							        <li class="item"></li>
							        <li class="item"></li>
							        <li class="item"></li>
							        <li class="item"></li>
							        <li class="item"></li>
							        <li class="item"></li>
							        <li class="item"></li>
							      </ul>
							      <script>
							        $(function() {
							          var wall = new freewall("#container");
							          wall.fitWidth();
							        });
							      </script>
							    </body>
							    </html>

Options

All options are parameters of Reset method

  • animate : boolean

    True to make block move with animation

  • block.flex : boolean

    True to made block can resize a bit to fit
    False of block has class 'fixed' to keep size of block can't change

  • cell.width : number

    The width of unit, base on it will build grid container.

  • cell.height : number

    The height of unit, base on it will build grid container.

  • fillGap : boolean

    True will made layout without gap.

  • gutter : mixed

    The spacing between the blocks. Default is number, but can set it with 'auto' value.

  • selector : string

    Get all blocks to made layout

  • Example 

    								
								    var wall = new freewall('.free-wall');
								    wall.reset({
								      selector: '.brick',
								      animate: true,
								      block: {
								        flex: true
								      },
								      cell: {
								        width: 150,
								        height: 150
								      },
								      fillGap: false
								    });

Events

  • onResize

    Registry callback when browser resize.

  • Example 

    								
								    var wall = new freewall('.free-wall');
								    wall.reset({
								      selector: '.brick',
								      onResize: function() {
								        this.fitWidth();
								      }
								    });

  • onSetBlock

    Registry callback when arrange block.

Methods

  • fitHeight(height)

    Made the layout fit with limit height

  • fitWidth(width)

    Made the layout fit with limit width

  • fitZone(width, height)

    Made the layout fit with limit width and height

  • fixSize(object)

    Set a block with special width or height
    Example 
    								
								    var wall = new freewall('.free-wall');
								    var dna = $(".free-wall .flex-layout");
								    wall.unsetFilter();
								    wall.fixSize({
								        block: dna,
								        width: 150,
								        height: 150
									});
								    wall.fitWidth();

  • reset(options)

    Config options for render layout.
    Example 
    								
								    var ewall = new freewall(demo);
								    var demo = $(".example");
								    ewall.reset({
								      selector: '.cell',
								      animate: false,
								      block: {
								        flex: 1
								      },
								      cell: {
								        width: 150,
								        height: 150
								      },
								      fillGap: true
								    });
								    ewall.fitWidth(cwidth);

  • setFilter(selector)

    Fillter blocks to show

  • unsetFilter()

    Made all block to show
